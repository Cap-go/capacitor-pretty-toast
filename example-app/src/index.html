Capgo Capacitor Pretty Toast

Native island-style toasts with the RN-shaped API.

This demo exercises queueing, force interrupts, live updates, promise flows, raw SVG icons, remote images, and data-URL icons from one page.

API `toast.*`
Targets iOS, Android, Web
Modes Symbol, SVG, URI

Shared Controls

These values are applied to most demo toasts.

Variants

Shorthand calls using the same defaults as the React Native package.

Queue and Force

Normal queueing, then an interrupt using `force: true`.

Promise Flow

`toast.promise()` morphs a loading toast into success or error.

Icon Modes

`icon` supports symbol strings and raw SVG. `iconSource` stays URI-only.

Event Log

Press, action, queue, and promise activity is echoed here.
